# wings30x2

Use [the wiring diagram](https://github.com/codefresco/wings30x2/blob/main/diagrams/wiring.jpg) in `./diagrams` to wire the STM32F411 controller to the pcb and the usb ports. Thinner wire such as AWG 30 or AWG 28 with silicone insulation wire is preferred for better fit.

The stand offs on the case are for M2x4 heat inserts. To fix the controller to the bottom plate CA glue or thin two part epoxy works well.

Firmware is vial-qmk and key binds can be edited using (vial)(https://get.vial.today/download/)
